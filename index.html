Conferences, Podcasts and Videos

  • Wordcamp Boston —Excellent web design conference, although it is strongly Wordpress themed there are many talks that cover general web design concepts. Excellent group of speakers and attendees, in the open-source Wordpress fashion speakers volunteer and the conference is very inexpensive, usually $40 for two days including a great t-shirt. Most of the Wordcamps around the world film their talks so you can watch them on Wordpres.tv
  • Podcasts: The Web Ahead —Jen Simmons topic focused interviews with famous web people The Big Web Show with Jeffrey Zeldman. More chaty and philosophical than Jen's show.
  • An Event Apart Boston —A very prestigious web design conference. Started by Jeffrey Zeldman. A good way to learn way where the industry is headed. Expensive. Some free-to-watch videos from the conference.
  • Build Conf Videos 2013 —Very philosophical talks from famous web designers.
  • Shop Talk Show —Interviews with web personalities Hosted by Dave Rupert and Chris Coyier.
  • Web Directions Videos —Videos from the Web Directions conference.
  • Five Simple Steps Videos —Aggregation of web conference videos.

About the Course

An introduction to the planning, organizing, designing and building of usable, accessible and beautiful web sites using industry-standard design methods and modern, semantic development techniques. Students will be exposed to a user-centered design process that will aid them in balancing aesthetics with user experience and will learn to build their designs for the web using HTML5, CSS3 and jQuery. Prerequisites: CDGD-220.
