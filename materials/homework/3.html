Create a portfolio for a visual artist you admire (not yourself). Site should contain images of at least a dozen of the artist's works. If you obtain the images online they must be properly attributed. Due by the start of next class.

Adapted from Saul Baizman's Image Gallery assignment.

Goals

  1. Practice formatting images for the web
  2. Practice using CSS to layout images and captions
  3. Practice building a multi-page HTML website

Deliverables

A visual portfolio website synced to GitHub and linked to from your Student Page. Should contain at least:

  1. Twelve images captioned with the title of the artwork, year, medium, and dimensions (if applicable)
  2. Each image should be a thumbnail (less than 150kb) wrapped in a link to a larger size image
  3. A separate HTML about page describing the artist or containing their biography. You may copy an existing biography if properly attributed and reformatted cleanly in HTML.

CSS styling including font choice, type sizes, type weight, color, padding, margins, image floats, image sizing and other techniques taught to-date.

Site should be usable and well designed on a variety of browser platforms included small-screen laptops, phones, tablets and large monitor desktops.
