Homework #7 —CSS Zen Garden

  • Course Home

According to its website, the CSS Zen Garden is “a demonstration of what can be accomplished through CSS-based design.” The website is both a gallery of inspiring examples of web design and a challenge to would be CSS masters. The rules of the challenge are as follows:

  1. Download one HTML file and one CSS file (provided by the site)
  2. Modify the stylesheet in any way you wish, but not the HTML
Due by the start of next class.

Background

The CSS Zen Garden was created in 2003 by Dave Shea during an exciting time of transition within the web design community. In this same year Jeffrey Zeldman released the seminal book Designing with Web Standards promoting the use of semantic HTML markup, separation of content from presentation and a denouncement of table-based layouts. CSS Zen Garden proved to skeptical web designers that beautiful layouts using CSS were possible.

For the ten year anniversary of the Garden, in 2013, Shea, re-opened the garden to submissions. He updated for the first time the HTML, using modern best practice including new tags such as header, section, aside, footer, nav and adding ARIA roles. He ported all of the old (official) CSS themes to use the new markup and re-launched on GitHub. Browse through the hundreds of themes created since 2003, get inspired, and start coding.

Adapted from Saul Baizman's CSS Zen Garden assignment.

Goals

  1. Gain an understanding of the power and vast limitations of influencing visual design by only changing CSS.
  2. Step into a not yet forgotten piece of web design history.

Deliverables

Your task is to create a CSS Zen Garden theme. It is recommended that you sketch out your ideas on paper before diving into the CSS.
  1. Style your own version of CSS Zen Garden modifying only the CSS
  2. Link to your Zen Garden from your Student Page

Revisions

The following are suggested incremental steps towards moving your projects forward. They do not have a fixed due date, manage your time appropriately.

  1. For your Client Site create all of your HTML pages. Start adding textual content and images. No need for CSS styling at this point.
