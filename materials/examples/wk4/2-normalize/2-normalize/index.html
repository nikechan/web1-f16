Normalized stylings

This page is identical to the default stylings page except that we have included an additional stylesheet in our html created by Nicolas Gallagher which normalizes (or resets) the browser default styles to a set of universal styles so that our designs will look a bit more uniform across browsers.

By the way, Eric Meyer originally coined this idea and called it a reset stylesheet at the time.
